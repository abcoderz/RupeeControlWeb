Rupee Control FAQ


General questions

What is Rupee Control app?

Rupee Control App is track personal Income and Expense offline in your mobile.

My income, expense data secure with this app?

Yes, All your income, expense data save locally in your mobile. App does not copy, backup any data to server. App also have fingerprint lock to secure your data.
How to use application

How to add income?

step 1 : Click on +(plus) icon top,left of app main screen. step 2 : select date time, enter income amount, select category, add note (optional) step 3 : click on save button

How to add expense?

step 1 : Click on -(minus) icon top,right of app main screen. step 2 : select date time, enter expense amount, select category, add note (optional) step 3 : click on save button
